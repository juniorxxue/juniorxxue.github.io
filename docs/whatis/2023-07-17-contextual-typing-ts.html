Contextual Typing In TypeScript

Posted on July 17, 2023

In TypeScript Handbook:

Contextual typing applies in many cases. Common cases include arguments to function calls, right hand sides of assignments, type assertions, members of object and array literals, and return statements.

Simply speaking, contextual typing is one form of type inference in the use of contextual information. In this post, I will elucidate them by borrowing rules from Understanding TypeScript.
